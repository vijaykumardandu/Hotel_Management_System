# Hotel Management System (Python, Tkinter)

A simple desktop application to manage hotel guests using Python and Tkinter.

---

## Features
- **Check-In Guests:** Add guest name, phone number, and room type.
- **Show All Guests:** View all current guests in the system.
- **Check-Out Guests:** Remove guests from the system by name.
- **Interactive GUI:** Uses Tkinter for an easy-to-use interface with message boxes.

---

## Technologies Used
- Python 3.14
- Tkinter (built-in GUI library)

---

## How to Run
1. Download the project folder (or ZIP file).  
2. Make sure Python 3.14 is installed.  
3. Run the app by double-clicking `hotel_management.py` or using the terminal/command prompt:  
   ```bash
   python hotel_management.py
